# Web_Map Application

Приложение предназначено для мониторинга событий и отображения их на карте.
Проект реализован с помощью фреймворка Django.
В настоящее время не поддерживается. Фронтенд переписан на Vue3 [здесь](https://github.com/prosto-kosmos/Web_Map_Vue3)
